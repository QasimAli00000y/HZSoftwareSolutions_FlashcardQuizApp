# HZSoftwareSolutions Flashcard Quiz App

A **cross-platform Flutter application** that allows users to study and test their knowledge using flashcards. Designed for simplicity and engagement, it works on Android, iOS, web, macOS, and Windows.

## Features

- Create, edit, and delete flashcards
- Organize flashcards into categories or decks
- Quiz mode to test knowledge with multiple-choice or written answers
- Track progress and review incorrect answers
- Responsive design for mobile, web, and desktop
